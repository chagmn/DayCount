# DayCount
디데이 계산 어플(ver. Swift)

심플하게 만들어 본 디데이 계산 어플리케이션입니다.
+버튼을 누르면서 디데이 항목을 추가할 수 있으며 오른쪽으로 Swipe를 통해서 항목을 삭제할 수도 있습니다.

같은 어플을 SwiftUI로 만든 버전도 있습니다!!
